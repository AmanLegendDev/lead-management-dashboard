# Lead Management Dashboard

Mini CRM-style full-stack dashboard built as part of hiring assignment.

## Features
- Admin login (basic auth)
- Leads table with search, filter, pagination
- Lead details page
- MongoDB Atlas database
- Backend APIs (Next.js API routes)
- Analytics dashboard
- Fully deployed app

## Tech Stack
- Next.js (App Router)
- MongoDB Atlas
- Mongoose
- Tailwind CSS
- Faker.js (dummy data)
